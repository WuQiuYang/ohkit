# Measure
 
可监听dom各项尺寸变化的高阶组件

## Basic

<Playground __code={codeDemo}>
{() => {
    return <div>
        <h3>调整窗口大小查看效果：</h3>
        {<Measure client>{({measureRef, contentRect }) => {
            return <div ref={measureRef} style={{backgroundColor: 'antiquewhite'}}>
                width: {contentRect.client?.width} - height: {contentRect.client?.height}
            </div>
        }}</Measure>}
    </div>
}}
</Playground>

``` tsx
<Measure client>{({measureRef, contentRect }) => {
    return <div ref={measureRef}>{contentRect.client?.width}-{contentRect.client?.height}</div>
}}</Measure>
```
