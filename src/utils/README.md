# @ohkit/utils

工具函数集合库，提供多种实用的工具模块

## 功能模块

- **async**: 异步操作工具
- **auto-reporter**: 自动报告工具
- **debug**: 调试工具
- **dom-helper**: DOM 操作辅助工具
- **json-helper**: JSON 处理工具
- **platform**: 平台检测工具
- **prefix-classname**: 类名前缀工具
- **react-helper**: React 辅助工具
- **sse**: Server-Sent Events 工具
- **tree-helper**: 树形结构操作工具

## 安装

```bash
npm install @ohkit/utils
```

## 使用示例

### 按需安装特定模块包
```typescript
import { dfsTree } from '@ohkit/tree-helper';
import { debug } from '@ohkit/debug';
```

### 完整安装，按需引入
```typescript
import {dfsTree, debug} from '@ohkit/utils';
```
